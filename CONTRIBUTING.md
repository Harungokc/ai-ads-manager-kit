# Katkıda Bulunma

## Nasıl Katkıda Bulunulur?

1. Bu repoyu fork'la
2. Yeni bir branch oluştur: `git checkout -b yeni-ozellik`
3. Değişikliklerini yap
4. Commit at: `git commit -m 'Yeni özellik: ...'`
5. Push at: `git push origin yeni-ozellik`
6. Pull Request aç
